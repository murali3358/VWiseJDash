Where do you access Bulk Posting?   In the top right hand corner of the Service/Activity Posting Screen.  Locate the Service Activity Bulk Posting Button in the image below.

Please note that one CA and one MS have been selected (the box in the first column of the list box is checked.)  This action tells the system which CA & or MS should be the template record(s) used moving forward. 

In the example image above you can see Assessment of Need (CA) and Basic Needs Met (MS) are checked.

In this Service Plan there is a date in the FollowUp column for Assessment of Need.  Please note moving forward that the bulk posting feature ONLY interacts with the Completed Date.  The FollowUp column dates will NOT become a part of the template record. 

 

Now let’s look at the bulk posting working screen next. 

This screen has 3 distinct sections:

·         Filters that can be used to search for applications

·         The Grid Box where applications are called up (based upon the filters set by the User)

·         Communication Board that tells the User the condition of the template record(s) in the selected Intake in the Grid Box.
