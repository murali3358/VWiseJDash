USDA Meal Reimbursement Report Screen (CACFP)

The USDA Meal Reimbursement Report aggregates meal counts and reimbursement values in order for your to submit your CACFP meal reimbursement report online. 

  • Site allows you to run one report for all centers, or select one or multiple centers/classrooms
  • Meals filter should always be ALL.  This allows you to see if a child's intake was inadvertently tagged anything other than FREE.  It's a great audit while you're reviewing your report.  All HS children are reimbursed at the FREE rate when enrolled in a HS program.
  • Funding Sources is a filter you use to ensure you are only looking at meal records that should be included in the report.
  • Report From Date is where you put the open and close date for the report you are running.  (ie, if you are running a report for January you would use the first date and the last date for the month of January.)  The system will only include meal data that qualifies in the filtered date range.
