Detail Attendance Report Screen

Detail Attendance Report will generate a detailed or summary report of attendance absence codes within the given report from date and to date you set in the filter. 

Present days (summary total) and Meal Counts (summary total) will also be presented, by child.

Generating the Report Type filter Consecutive Days Absent with a ZERO in the number of consecutive days will allows the User to review all detailed attendance (present, absent and meals) for the child(ren) in the given date range.
