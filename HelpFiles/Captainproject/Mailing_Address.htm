Enter Mailing Address in the Client Intake Screen

If the mailing address is different than the physical address for the application you are completing, enter the mailing address in the Intake Details Tab. 

This feature allows Users to create accurate mailing labels using the mailing address as opposed to the physical address where mail is not received.  Do NOT complete this section if the mailing address is the same as the physical address. 

Access the working screen by clicking the Mailing Address button below.
