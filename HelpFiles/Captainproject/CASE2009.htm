System Administrators work with Program Managers to create a list of criteria that qualify a client for services. Individual Group Descriptions are developed based on the program specific eligibility criteria.  Very specific questions are associated to the group to determine program eligibility.

This is the hub of the Logic Engine. Conditions define presumptive eligibility leveraging every data element within the agency intake with the ability to include it in the eligibility criteria for individual programs.

  • Select data fields from the Intake database to help automate the eligibility determination process, or
  • If your Program’s eligibility is not determined by one of the options in the master index table panel, the System Administrator will need to make it one of the available questions in the Custom Fields Entry panel. Once the questions has been added in the Custom Fields Entry panel the User will now go back to the Program Definition screen and select the data fields to determine eligibility.
  • All eligibility criteria can be fed into the Service Integration Matrix/Benefits Bank (see ‘Include in SIM’ check box in Program Definition Screen) as well. Doing this enables one program to complete an Intake and have the System do a preliminary determination if the client is eligible for other programs in the agency.  

Search for the Hierarchy you want to work with

Enter the specific hierarchy code you want the system to select and click the search button.  When the system locates that hierarchy in the list it highlights it in blue and presents an arrow in the left hand side of the row.  This tells the User where the system is working during  ADD and/or EDIT mode. 

*NOTE - When editing exisiting records, it is important to ensure you have selected the correct hierarchy you want to work with to avoid editing a group description in the wrong hierarchy.

When the system is locked in on a specific hierarchy the screens on the right hand side load with any group descriptions associated to that hierarchy.  Let's look at an example image.
