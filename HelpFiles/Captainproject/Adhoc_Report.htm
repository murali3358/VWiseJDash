Adhoc Report Screen

Think of Adhoc Reporting as a blank piece of paper where you organize data fields from the system into your own, personal report to review the data!

The screen has 4 sections:

  • Report Filters
  • Table List
  • Column Descriptions
  • Report Design Area (the blank piece of paper)

Report Filters - filters allow the User to instruct the system to do certain things when generating the report.

Table List - This section presents the Tables available in the system and a meaningful description of the area/screens in the system that host the fields in the tables. 

Column Description - this section presents fields available in the table list.  The fields that are displayed are based on the selected Table in the Table List.

Report Design Area (the blank piece of paper) - this is where you design your report.  Fields that are selected (green check mark) as the Column Headers in your report will appear here. 

How to Generate an Adhoc Report

Locate and Click the  button to access the working screen to complete the setup of your report.

 

  • Report Name - Type in the name of the report
  • Header and Footer check boxes - check or uncheck these boxes based on the filters you want the system to use when building the report
  • Header Title - Type in the header title you would like at the top of the report page. 
  • Page Orientation - Select the orientation you want the report to be designed in.
  • Click OK - this initiates the report generation. 

Once you click OK, the Report Viewer will open and you will see a blank piece of paper while the system builds and designs the report.  Once the system has completed that you will be able to review your report.  Let's look at an image of the top part of the Report Viewer. 

**Note - at this point in the process the system has built and designed your report; however, it is NOT saved anywhere.  It is held in the report viewer only.

  • We can see the report has 394 pages. 
  • You can click on the standard arrows to move throughout the report or you can use the scroll bar on the right hand side of the viewer. 
  • You can use the search function to find data in the report
  • You can save the report by selecting the file format option you want
  • While using the search function, you can refresh the report back to its original presenation by clicking the  button.

 

How to Save an Adhoc Report

Locate and Click the save button at the top of the Report Viewer.

When you click the arrow to access this drop down you will see the file format options you have to select from when saving your report.  The system is now working outside of the system, communicating directly with the PC you are working on.  You will be prompted to name the report and save it on your PC.  The User determines where the report is saved, not the system. 

  • Excel - creates an excel workbook with the report in the sheet1 position
  • PDF - creates a pdf document of the report
  • WORD - Creates a WORD document of the report.

Any continued activity with this report is now outside of the system!
