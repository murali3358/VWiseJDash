CA Definition Tab in the Master Tables Definition Screen

Critical Activities are a key part of Service Plans, ROMA Goal Management, and Reporting.  When building Service Plans you select the critical activities you want associated.  When Users select a specific Service Plan whatever critical activities have been selected will be available for the User to interact with.

During report building, critical activities can be selected to be included in Group Definitions for report building.  This eliminates the need to select individual services every time you want to build a report. 

In order for any User to select or interact with a critical activity it must be defined here first. 

 

  • Critical Activities are defined in this Tab. 
  • The main toolbar in the Tab controls the functions available to you while you're adding and editing. 

Add a Critical Activity in the CA Definition Tab

Locate and click the ADD  Button in the Tab toolbar to access the working screen to add a new critical activity.

  • Service Code - A unique User assigned code that identifies the critical activity in the system when used in reporting.
  • Active Check Box - Check this box when you want the critical activity to be visible so a User can select it.
  • Description - A brief description of the critical activity.
  • Click Save

 

Edit a Critical Activity in the CA Definition Tab

Locate and Click the EDIT Button in the Tab toolbar to access the working screen to edit an existing critical activity.

Make the necessary edits you need and then click SAVE.  Do NOT delete critical activities that have been used.  Uncheck the Active Checkbox and then Users can no longer see the critical activity to select it.  It sleeps quietly in the background prepared to deliver historical data as needed.

Copy from Servicein the CA Definition Tab

Locate  to have the system copy an existing service into an ADD screen to create a new one.  This helps Users save time by just edit small changes in an exisiting service. 

Highlight the service you want to copy, then click the Copy from Service button.  The system will then open a new ADD screen for you to work in.
