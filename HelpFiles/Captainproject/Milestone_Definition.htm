Milestone Definitions Tab

Milestones or Outcomes are custom designed in this Tab. 

Add a New Milestones or Outcomes to the Milestone Definitions Tab

  • Click Add in the Standard ToolBar to access the Add Screen
  • Service Code - Design a unique alpha/numeric code to identify the Milestone you are creating.
  • Active Check Box - leave this box unchecked if you don't want Users to be able to see the Milestone/Outcome until you are ready.  Once you check the box, all Users will see it.
  • Type - Select either Milestone or Outcome
  • Description - Enter a meaningful description that will allows Users to identify the Milestone/Outcome while working in the system.

 

Edit Milestones or Outcomes to the Milestone Definitions Tab

  • Select/Highlight the Milestone/Outcome you want to edit.
  • Click Edit in the Standard ToolBar in order to access the Edit Screen
  • Update/Change fields you want to edit
  • Click SAVE
