Ranking Categories Tab

Ranking and Risk Assessment is a feature in the system that will look at specific point-based criteria and determine where applications rank within the Category.  You can choose from standard fields in Client Intake, Medical Emergency, and also custom fields. 

This feature can be used to Rank Head Start children for selection and enrollment.  It can also be used to complete an automatic Pre-Assessment of clients and their needs in order to determine if there is a need to move on to a Full Assessment. 

Ranking Categories are built and maintained through the Standard Toolbar at the top of the Master Tables Maintenance Tab.   

Add Ranking Category in the Ranking Categories Tab

Click  in the Standard Toolbar above the Master Table Screen Tabs in order to access the ADD Screen.

  • Code - Assign a unique code.
  • Description - Enter a meaningful descriptions that Users will see.
  • Head Start Ranking Category - assign the ranking category you want the system to use.

Add Point Range to a Category  in the Ranking Categories Tab

Locate the Ranking Category row that you want to work with.  Look at the right hand side of the row and locate the ADD COLUMN.  Click the  on that row to open the Point Range working screen.

  • Code - Assign a unique code
  • Description - the name you want to assign to the Point Range
  • Points Range - the number of points, based on data entry for assigned questions that must be achieved to fall into this Point Range.

Once each Point Range is defined, they will be in the bottom section of the Ranking Categories Tab.

Define the specific fields to be used in the Ranking Categories Tab

Locate the Ranking Category row that you want to work with.  Look at the right hand side of the row and locate the DEFINITION COLUMN.  Click the  on that row to open the Point Association to Field working screen.

  • Highlight the Response you want to work with.
  • Double Left Mouse Click the Points column for the highlighted response.
  • Enter the point value for the response.
  • Click SAVE.

You can assign ALL point values for ALL Responses before clicking SAVE.
