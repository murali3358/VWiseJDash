View/Edit Agency Switch Button

Use this button above to access the agency switches.  It is found in the lower right hand corner of the Hierarchy Definition Screen.  You interact with the check boxes to set or unset the switch.  A checked box activates that switch. 

 

  • Services Inquiring From Programs - XX
  • Show Case Manager Combo in Case2001 - XX
  • Edit Zip and City in Case2001 - checking this box allows the User to change the zip and city field in an existing application that has been saved.
  • Clear Reports from CAPReports folder - checking this box clears all reports placed in the capreports folder.  It is recommended that you do NOT check this box in order for Users to have access to reports previously run.  Once you have used the system for a while you will know better whether to return and check this box.
