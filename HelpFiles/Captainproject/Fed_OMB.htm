Fed OMB Chart (OMB) in Master Poverty Guidelines Screen

The Fed OMB Chart is required in the system in order for household income to be verified.  Income details attached to an application are compared to the Fed OMB Chart in the system to plot the OMB % of the household for program eligibility.  This field is also resourced when the User runs the Case Demographic Report.

While a system adminstrator can opt not to create any other master poverty guideline charts in the system, Fed OMB is mandatory.  

Annually the Poverty Guidelines are released from the federal gov't and must be input into the system.  If the Fed OMB chart expires, Users will not be able to verify income for any application outside the date range of the last chart built in the system.  If you find you are waiting for the federal gov't to release the next poverty guidelines, extended the end date of the existing OMB chart to ensure that income can continue to be verified while you wait.

****Note - if the poverty guidelines are released with a retro-active date (in comparison to verificaition dates already used in the system) the System Administrator should run a report to identify applications with verifications dates that are impacted by the change so that those applications can be revisited and updated.   This will ensure that all applications were verified with the correct poverty guidelines chart being resourced.
