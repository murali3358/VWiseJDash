Master Table List

The Master Table List is a section that the system administrator can use to run reports on different areas of the system they have built.   This allows a full review of all tables, associations or lists. 

The following Report Types are available to select from:

  • Agency Table LIst
  • Case Demographic Associations
  • Performance Measures Associations
  • Crititcal Activities
  • Milestone List
  • Zip Code
  • Field Control Maintenance

In the Agency Table field you can select ALL or just one specific table for the report.
