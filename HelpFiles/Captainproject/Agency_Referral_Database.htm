The Agency Referral Database manages the list of internal and/or external referral entities that you want Users to interact with.  Easily manage your entire agency referral list within CAPTAIN.  A service activity posting feature for managing referral data at a very high level.  See Referred TO/FROM details in the Service Activity Posting Tab. (See Add a Referral in Service/Activity Posting )

View a Referral Entry in your referral database

Use Edit Mode to view individual entries in the agency referral database. 

1.  Left Mouse Click the database entry you want to select.  The blue highlight indicates which entry in the referral database the system you have selected.  The system will now interact with your continued selections relative to your selection.  The arrow on the left hand side of the row is an additional indicator of which entry has been selected.  

2. Click EDIT  to open the Edit Screen to view or edit the specific information entered for the agency referral entry you have selected.

3.  An entry that is inactive will be displayed in RED FONT.   You can change back and forth from active/inactive in EDIT MODE by checking or unchecking the active box. 

Add a New Entry to the Agency Referral Database

Click ADD at the top of the Agency Referral Database Tab to open the screen where you can input the details for the agency you want in the database.  The system will automatically assign a Code to the entry you are working with. 

  1. Contact Information should have the appropriate contact person and phone number to use when communicating. 
  2. The Service Button allows you to access the services table and associate specific services to the referral agency.
  3. Do NOT delete referral agencies from the database.  Make them inactive.  Then when you don't want Users to have access to a specific referral agency it won't be available, however, you will be maintaining important historical information in your system. 
  4. Click SAVE once you have completed all appropriate field entries.
