Ranking/Risk Assessment Report  

This report will generate a ranking report of all Client Intakes in the system.  Any Client Intake that interacts with the specific data designed in the ranking category, in the Master Table List, will be in your report. 

Set your filters to run the Ranking/Risk Assessment Report in the screen below.

  • Ranking Category - select the ranking category you want the system to use
  • Process By - set the filter to determine which data field the system will use and what date range
    • Date Add - date the Client Intake was saved in the system
    • Intake Date - the date the User input for the Intake Date field in Client Intake
  • Sequence - the ranking order of all qualifying applications in the report
    • Name
    • Address
    • Rank Value (Ascending)
    • Rank Value (Descending)
    • Date (Ascending)
    • Date (Descending)
    • Hierarchy
  • Applications - select the filter you want the system to use
  • Report Format - select the filter in each row you want to use.  In addition, always check the Update Ranks in Case Master if you want the system to keep pace with the current data entry completed by all Users.
