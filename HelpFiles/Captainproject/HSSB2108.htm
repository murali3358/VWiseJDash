Attendance Sheets Report Screen

Use this report screen to generate attendance sheets for staff to use to track attendance and meal counts.

 

  • Children Age - leave this as the default to ensure all enrolled children appear on the report
  • Active/Inactive - you can leave it as the default, BOTH, or select ACTIVE. 
  • Week Starting Dates - You can create one full month of attendance sheets if you use all 5 fields in the same report run. (This is not recommended as enrollment lists change every day and that will render future attendance sheets useless.) 
  • Site - ALL sites will produce attendance sheets for the entire program.  Select Multiple Site/Classes to select just the one you want to work with.
  • Base Ages on - Today's Date is the default and should be used.
  • Print Adult Meal Info?  - Select Yes if you want a place on the attendance sheet to track adult meals.  Select No if you don't need that on the report.
  • Print a blank line between each Child -  check this box if you want an additional blank line between each child for extra room.  Selecting this report addition may push your attendance sheet on to two pieces of paper.
