Use the system standard ADD CHANGE and DELETE buttons below in order to work with this table.

 

 

This table screen is referenced by some as the ‘MOR  table’  (Margin Over Rack) or Harbor Price Table.

This screen allows the authorized user to enter the date, terminal, oil and kerosene price per gallon for these deliverable fuels by terminal.

When the User posts a delivery date and selects the vendor when posting an invoice the system will go to this table and get the price per gallon for that fuel type. The objective of doing this is to afford the opportunity to use the lesser of the retail price versus the harbor plus Differential prices for CT Agencies.

The entry of the data is done underneath the list box in the fields shown above.
