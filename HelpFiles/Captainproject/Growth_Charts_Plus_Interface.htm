Growth Charts Plus Interface Report Screen

The Growth Charts Plus Interface Report feature allows the User to create growth charts for the child's binder.  No more plotting heights and weights and hand drawing each and every one.  This features interacts with specifically tagged tracking tasks designed in the Master Track. 

  • Report Sequence - this filter will sort the data by name or App#
  • Active/Inactive - Active, Inactive or Both
  • Site - select the site(s) you want to include in the report run
  • Funding Sources - select ALL or Selected if you want one or more but not ALL
  • Selected Tasks - the specific tracking task that you asked the system to use when creating the charts (Master Track)
  • Head Circumference - the specific tracking task you asked the system to use when creating the charts (Master Track)
  • Children Age - enter the age range in the fields to filter the data records and charts you will receive in the report run
  • Base Ages on - the default is today's date, select the filter you want the system to use
  • BMI - this filters allows you to limit the data records and charts you will receive in the report run based on the BMI range you set.  (i.e. - target a specific % group that you want to work with.)
  1. Generate the Grid Button will create the grid of enrolled children with available data records for charting.
  2. Generate the Charts Button will do just that
  3. Chart Preview Button allows you to look at the charts prior to printing them out.

You must work with the steps in the specific order above. (1,2,3) You first need to generate the grid of children the system will lock in on.  Then the system will use the available data attached to each child in the grid and create the charts.  Then you can preview the charts that system generated. 

 Associating the tracking task to Growth Charts Plus Interface Screen

The specific height, weight and/or head circumference tracking question that you are using must be associated to the Growth Charts Plus Interface Screen in the Master Track.

Select the Master Track and then search for the specific tracking task you want to associate to the report screen.  In this example we will put our focus on the height task association.   In the edit screen below you can see that on the right hand side the system is asking the User to select the filter for the Growth Charts Plus Interface association.  What this does is tell the system to call up Track Task 0050, Height Screening #1 whenever the User is running the Growth Charts Plus Interface feature. 

The system will populate the height filter in the Growth Charts Plus Interface Screen with every association that has been established.  You are not limited to just one association.  (i.e. - in HS programs typically complete 3 height/weight screenings throughout the program year.  It would be wise to have 3 separate tracking questions in order to see the charting separately, by screening date.)
