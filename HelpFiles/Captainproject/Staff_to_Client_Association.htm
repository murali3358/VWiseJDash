This screen is where you have the option to select a staff member and associate clients to them. 

Select the staff member you want to work with by left mouse clicking it and making it blue.  The system is now locked on the staff record and all associations will be attached to that staff member. 

Set filters in the screen to minimize the client records you have to interact with.  For example the SHOW filter allows you to call up all clients, only associated clients, etc..  You can filter to a specific SITE, FUND, all kinds of options are there for you.  The best way to determine the correct filters for you is to do it a few ways and see what search results you get.  Then you'll know on repeat visits for additional data entry.

  • Click the PROCESS Button to generate a list of qualifying client records you can work with
  • When the gird box loads below the process button you will then be able to select individual client records and add a staff member association to that record. 

To Access the association check list, right mouse click the name of the client record row you want to work with.  You'll be presented with the working screen below.  Select the option you want to use.
