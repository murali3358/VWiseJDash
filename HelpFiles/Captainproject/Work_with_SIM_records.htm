Access the SIM Screen

Look over on the left at the Screen Tree and left mouse click on the SIM - Service Integration Matrix Screen.  The system will add this new screen to the working tabs on the right hand side.  You will have the ability to move back and forth from the Client Intake to the SIM Tabs without leaving the application the system is locked on. 
 
When you open up the SIM Screen this is what you will see.
 
 
Let's highlight a few things on the screen you will see:
  • Select Department Field allows the Users to change the information that is presented in the list box. 
  • The toolbar at the very top of the Tab controls the task you will perform.  The ADD button will allow you to run the SIM and create referral records for the Intake you are working with.  The SAVE AND CANCEL buttons are inactive until after you have created at least one referral record.
  • You can update Intake records at any time and run the SIM again to capture new or additional referral records.
  • The row the system is locked on is BLUE (system standard) and it highlights that selection below the list box.  (A second confirmation)
  • Peach Cells / Green Font indicate that a referral record can be created based on data in the application.  This could or could not reference the questions section of intake details.
  • In this image above the screen is in VIEW mode.
Add/Work with referral record(s) to the application you are working with in the SIM Tab
Here is an image of the Toolbar at the top of the SIM- Service Integration Matrix tab.
 
 
Locate and click the ADD   Button in the SIM Tab Toolbar to have the system generate any available referral records for the Intake you are working with.  If there are any referral records to generate the system will allow the User to GREEN CHECK MARK the RED X in the SEND column of the screen.
 
Here is an image to show an example of that.  To change the RED X to a GREEN CHECK MARK the User must left mouse click the cell with the RED X. 
 
 
  • Green Check in the Send Column - two referral records have been generated for us to work with.  (they have not been saved at this point)
  • Decline -  this column allows a User to communicate the condition of the referral at any time.
  • Date - date the referral record was generated
  • Referred By - User the generated the referral
  • Contact Name - the name of the person who will assist with the referral for the specific program the referral was generated to

The SIM Toolbar has now changed!  The system follows you as you interact with feature functions.  Let's look at the new toolbar after you have generated available referral records.

  • The ADD function is no longer available. (in moving forward with the SIM and this application you will work in EDIT mode)
  • DELETE, SAVE, and CANCEL have now activated.

If you do nothing else and close the SIM Tab the referral records that have been generated will be lost.  When you click SAVE, the ADD function is replaced with the EDIT function.  Let's look at the toolbar image.

If you determine you don't want a record, you can delete it.  You can delete a referral before you clicked SAVE or after you have clicked SAVE.  Please note this is a permanent delete and the date original data and date reference will be gone.  You can, of course, generate the referral record again but the date reference will be different if it happened on a different date and if another User interacted with the Decline column that will be gone. 

Any User with access to the system can generate a referral record or interact with the DECLINE column to update communcation about the condition of the referral. 

Print a SIM Referral Letter

Here is an image of the Toolbar at the top of the SIM- Service Integration Matrix tab.
 
 
Locate and click the PRINT  Button in the SIM Tab Toolbar to access the print screen. 
 
 
Check the SIM Referral Letter box and then click the  button.  This will allow you to access the referral letter and print it.
