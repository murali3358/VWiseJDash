This report shows the User the clients that have moved their funding so that when you want to tie out your financials you can see the movement of the clients and if any errors are present.  (ie, a client was paid and for some reason a User removed the Vendor and the payment is hanging out there.)

This report should be run weekly, starting in January of the program year until the end of the year. 

Update filter:   once this report is run in Update, the system moves all the records to the appropriate funding source. 

No Update:   same report, without actually moving all the records.  Run this report first, make changes, then run the update.
