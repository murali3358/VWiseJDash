Users run this report to get a list of applications there were certified and a User changed information in the household.   It is also used to run a list of applications not yet certified (Intake Date but no Control Card). 

Start Date:  Always input the first date of the program year.  This ensures the system will include all applications that can be considered.

End Date:  Use the current date that you are running the report. 

Sort By:  set the filter for the system to sort the report by Name or Application Date

Process Report:  The default is ALL.  Select the filter if you don’t want both.
