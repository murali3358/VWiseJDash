This is a very simple report that presents a list of Vendors with details.

Terminal:  All Terminals is the default.  CT only has Terminal 1 at this time.

Full List:  gives all demographics on the Vendor.  Condensed:  gives less information, no address no phone. 

Contract Agreement:  Vendor Maintenance has a check box where a User can confirm that the contract has been returned from the Vendor.  If your agency uses that methodology to track vendor contracts you can run this report to see that list.
