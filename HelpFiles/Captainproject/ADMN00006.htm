Users cannot create a new Table independent of CAP Systems. Development and implementation of all new Tables is managed by CAP Systems and added into your System when tested and ready to use. Technical support is available for development discussions through the Help Desk.

Agency Table Codes are managed lists that determine what options are available in the drop down fields throughout the system that Users work with. This screen allows you to:

  • ADD  and/or EDIT drop down options in a specific agency table.
  • Make the option INACTIVE if it is no longer used. 
  • You cannot delete a table from the system once it has been added.

  • The Application Field at the top of the screen allows you to see all the tables in the systemor only tables for a specific application.  (ie - selecting Case Management creates a filter so that you are only able to select from the tables that are available in the Case Management Application.) 
  • Sort On by Description is alpha order by Cell 7.  Sorted by Code is numerical order by Cell 2.
  • The Agency Table Field is where you select the specific table that you want to work with. 

After you select the specific table you want to work with, the drop down options will appear in the grid box for you to work with. 

You ADD a new table code option on the row with the  on the far left side of the row.

When you move off of the row you have completed they system will tell you that the table has been updated.
