User Tree Structure Report

User Tree Structure generates a report that will you allow you to look at the structure of a User(s) Account.  This report shows you User privileges by application and whether or not the User has access to reports. 

You can select ALL Users or a specific User.

You can select ALL Applications or select a specific Application.

Generate PDF - generates your PDF report.  The system will present a SAVE AS screen for you to name the report before the system will generate it. 

PDF Preview - allows you to preview the report(s) you have generated.
