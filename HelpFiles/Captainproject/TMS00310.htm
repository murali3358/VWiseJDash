Incomplete Letter Posting Screen

 T he Incomplete Letter Posting Screen will help the User to create a specific incomplete letter for the selected applicant.  When printing an incomplete letter the font can be between 7-10 pitch.  You find this under the page setup button and 10 pitch is the recommended setting.

Before you can do an Incomplete Letter the applicant MUST have an Intake Date entered. 

If the User navigates to this Screen (from another screen) App# and Client Name will load with the applicant’s data.

Without navigating from another screen the User will need to enter the application number, client name or use the Browse button and do a Search to bring the application into the Screen. Using this method the application must have an Intake Date.

 

If the client has not had an incomplete letter previously the only button in the toolbar that will be enabled is the Add button.

 

If the selected client has had a previous Incomplete Letter the Add, Change and Delete buttons will be enabled in the toolbar.
