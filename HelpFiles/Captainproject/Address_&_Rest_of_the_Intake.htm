View Address and Intake Details in the Client Intake Screen

The bottom section of the Client Intake Screen is a display/view only area.  If a User has entered any information in these Tabs sections you will be able to select the Tab and review the information.

Add/Edit Address & Rest of Intake in the Client Intake Screen

Based on the field controls set in Program Definition by your System Administrator, complete the fields required/optional in the Address & Rest of Intake Tab in the Client Intake Screen.

Required fields are denoted by the RED Asterisk .  The User must complete all required fields in order to SAVE the Intake data entry that has been completed. 

Many of the fields in this section impact specific FUND reporting.  It is suggested that ANY field that is used for FUND reporting should be required to ensure 100% of the data is collected for the reports.
