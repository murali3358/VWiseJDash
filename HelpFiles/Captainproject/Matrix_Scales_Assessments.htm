The Matrix Scales Assessment tool is a feature built into the Case Management Enterprise that allows our customers to build Assessments from the bottom up.  It is a very high-level standardized measuring tool. 

Once additional assessments are completed the System then displays any dimension change (+/-) as a value.  The Caseworker can then expand beyond the standardized measurement and add subjective reasoning for these changes through the Reason Codes. 

The Matrix Assesment tool will allow an agency to assess the client(s) by asking several questions within each Scale.  The System will slot the assessment into a Benchmark based on the response.

As assessments are completed the System will compare the prior period's assessment to the current period's assessment.  In the end our customers have a tool that allows the caseworker to gather responses to questions while CAPTAIN measures those responses and presents them for review.
