Matrix/Scale Definitions Tab

The Matrix Scales Assessment tool is a robust feature built into the Case Management Module that allows you to build a comprehensive fully functioning case management Matrix.  This design allows you to build the Scales exactly as you use them at your agency.  

Build them.  Update them.  Change them.  You have 100% customization control.  From assigning a name to the Matrix to defining the Benchmarks (the scoring tools to measure client's progress) to defining the individual Scales (i.e., Money Management, Employment, Health Insurance, Housing).

The Matrix Assessment tool will allow a User to assess a client by asking several questions within each scale.  At the end the system will plot the scale into a benchmark based on the responses.  As assessments are completed CAPTAIN will compare previous assessments to the current one.

In the end the agency has a tool that allows a User to input responses to assessment questions while CAPTAIN measure those responses against prior period assessments.

The Matrix/Scales Tab offers high level defintion with three foundation steps:

  • Matrix
  • Benchmarks
  • Scales

These three steps must be completed in the bullet rotation.  Add in the Matrix.  Create your Benchmarks.  Create your Scales. 

After that you will then continue with the definition of:

  • Questions
  • Reasons
  • Outcomes

Deleting a Matrix or Scale from the database in the Definitions Tab

In order to delete a Matrix or a Scale from the database ALL access to the screen functions and reports must be removed for the Questions and Responses first.
