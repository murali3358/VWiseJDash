Add a New Service Plan to the Service Plan Admin Screen

Locate and Click the ADD  Button in the toolbar at the very top of the Service Plan Admin Screen Tab to access the working screen to design a new Service Plan.

  • SP Code - The system will assign a unique code to the new service plan
  • Active - when this box is checked the Service Plan is active and Users can see it
  • Allow Additional Branch - when this box is checked a User can see additional branches in the SP working screen.  These branches will be clearly marked and grouped according to the way they were designed.
  • Allow Duplicates - when this box is checked the User can add a subsequent SP.  Be cautious!  Duplicating a Service Plan will have some data records in one plan and other data records in another.  This feature is available when needed.  If you want ALL data records to be in the same service plan do NOT check this box. 
  • Desc - Type in a brief, meaningful description of the service plan that will allow Users to clearly identify the service plan when selecting it.
  • Service Plan Hierarchies - click the Hierarchy  Button in order to select the hierarchies that you want to have access to the service plan.
  • Branch Definition - Click in the branch Primary cell and NAME the primary branch of the service plan.
  • Branch 1-5 - if you want more than one branch in your plan you must identify to the system additional branches by naming them. 
  • Click SAVE SERVICE Plan to save the plan you have just designed.

Once you have saved a new service plan, you will find it at the END of the services plans.  It will be in RED font until you validate the plan. 

 

Edit an existing Service Plan to the Service Plan Admin Screen

Select (touch it, make it BLUE) the Service Plan you want the system to lock on and then Locate and Click the EDIT  Button in the toolbar at the very top of the Service Plan Admin Screen Tab to access the working screen to update/change an existing Service Plan.

 

 

  • Notice there is a Delete Check box on the left hand side of each row.  Once you check a box to begin the delete process for a CA or MS the DELETE  Button will appear on the far right hand side.  You can then delete rows one at a time. 
  • Be sure to SAVE SERVICE PLAN when you have completed all your edits.
