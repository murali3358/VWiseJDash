Client Tracking

This screen allows Users to manage, monitor and track comprehensive tasks for each child in the system.  This data also connects to the PIR Counting Tool where specific data can be monitored for the PIR Report day in and day out. 

If the enrolled child is supported by more than one funding source AND the data managed for both is different, you can manage both in here at the same time. 

Load SBCB - this button allows you to load in Should Be Completed By (SBCB) dates into the grid box for easy review. 

Component Field - Users can select a specific component area to filter out multiple questions they don't need to see during the record reviewing event.  The data is not removed, just filtered out so that you don't have to look around it or pass it by. 

Task - Users can filter to a specific task for a specific data record review event.  The filter ALL allows the Users to see everything and scroll through all the data records. 

Quick Post and Quick Edit - Users can resource class lists and post like information to several child applications at once.
