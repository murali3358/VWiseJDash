HS/AR Controls Tab in the Program Definition Screen

 

Additional Program Controls screen is for Head Start only.

Enroll Only Pre-Enrolled Clients: If Head Start uses this option all applicants must be pre-enrolled in order for the application to display in the enrollment screen.

Start Month / End Month:  Enter the program year start and end month for HS in these two fields.  This will dictate the month calendars that are available in the system to build the site schedule. 

Default Acct: N/A unless you are using the Accounts Receivable Application

Customer Type: N/A unless you are using the Accounts Receivable Application

Invoices Source: N/A unless you are using the Accounts Receivable Application

HS Age Control: this control functions in the HS Module when the system is calculating the age (in years) of the child for very specific Head Start functions.  You need to confirm whether your program's communities manage the Kindergarten Start Date with one date across the catchment area or different dates by school district. 

If it's one date across all communities, then you select the Zip Code filter and enter the universal start date.  You will find a date field for each school district in your system.  You must complete all school districts. 

USDA (CACFP) Meal Cost Categories: Accurately inputting the meal reimbursement rates ensures that the system will allow the USDA Meal Reimbursement Report to calculate the number of meals and the reimbursement amounts, by Site/Classroom. This pre-defined report replaces the need to manually calculate reimbursement amounts on the CACFP formatted report to the DOH.
