The Activity Summary filter allows the User to review application volume by FUND. 

 

The User will be able to review ALL application aggregated counts and then the system will aggregrate counts for Certified, Approved and Denied.

If the system has applications that have been taken but are not yet approved, that count will be included in the ALL application filter. 

Fill Data to Grid Button will present approved and denied applications only. 

The Weekly Financial filter give sthe User the cumulative numbers.  B1, U1 and R1.
