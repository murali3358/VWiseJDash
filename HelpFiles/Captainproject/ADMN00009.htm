Hierarchy Definition is where the System Administrator creates a three-level hierarchy that reflects, but is not limited to, the agency organizational chart. This is the foundation of how you will manage all your data. 

View the existing Hierarchy Definition

  • Left Mouse Touch to select the specific row you want to work with. 
  • The current selection is highlighted blue and there is a grey arrow on the left hand side of the row. 
  • The most important thing to note is always know where the system is focused.  Know where you are adding, editing, or deleting. 

  • This tab will add/edit the highest-level agency name position(s) as well as department or program level position(s).  How?
    • Departments are associated to the Agency Name you select. 
    • Programs are associated to the Department Name you select. 

The system will not let you assign a duplicate CODE number.
