The Appointment Schedule Report allows the User to review the scheduled appointments in the system. 

Users should run this report every day, end of day and be distributed to Intake Workers as appropriate.   Running and printing reports for more than the next day will not allow the Intake Workers to know their schedule is correct.  Changes made to appointment scheduling after the report has been run will not be reflected. 

The Date Range should be just one work day.  If a review of scheduling needs to be completed, the User can extend the date range beyond one day.  Those reports should not be distributed to Intake Workers because the scheduling could change. 

When the User selects the All Dates filter the report will present all dates included in the date range.

When the User selects the Today Through Future Dates filter the ereport will present from the current date and ahead.
