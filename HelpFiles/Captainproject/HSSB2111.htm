Average Daily Attendance Report Screen

The Average Daily Attendance Report aggregates all attendance data, compares it to the open funded days in the given date range of the report, and presents the average daily attendance for review.

Based on the report type filter used, you can see summary data or specific data for each classroom at each center.

 

  • Site - select the filter you want for your report
  • Report Type - Detail Summary gives you to maximum amount of information to review
  • Funding Sources - use this filter to run a report on just one fund (if you have more than one)
  • Report From Date - set the date range you want the ADA to be calculated on
  • Site for Home Based -  set the filter you need the report run to consider.
