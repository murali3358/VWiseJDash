Add Benchmarks in the Matrix/Scale Definitions Tab

Build in the five-point benchmarks that will measure incremental change for the outcomes in the Matrix:

  • Thriving
  • Safe
  • Stable
  • Vulnerable
  • In-Crisis

If your agency Scale doesn't use those benchmark labels, input your own!

 

  1. Select the Matrix you want to work with
  2. Enter the Benchmark Description
  3. Enter the Low number for this benchmark in the box
  4. Enter the High number for this benchmark in the box
  5. Click SAVE

Repeat this for each benchmark you need available in the Matrix you're building.

Edit Benchmarks in the Matrix/Scale Definitions Tab

In EDIT  mode you can make updates/changes to the Benchmarks already built in the system.

Click SAVE when you are finished to save your updates/changes.
