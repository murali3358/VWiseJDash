Add (Define) a New Agency Name hierarchy position to the system

Select the  button to access the Agency Definition Add Screen.

  • This screen allows you to define all of the general information as it related to the Agency Name design in the system.
  • Click SAVE when you have completed the fields. 

Assign a two digit unique code that will identify the agency in the system.   The short name is an abbreviated description that represents the Agency Name. 

All Intake - check this box to have all intakes interact with the SIM. 

In the top right hand section the following fields determine how your agency hierarchy field information is presented throughout the system.  Think of this as a standard default presentation:

  • Agency Name
  • Agency Representation
  • Client Name Format
  • Caseworker Name Format

Select the presentation style that will best suit the needs of your Users.
