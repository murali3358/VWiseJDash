Program Information Tab in the Program Definition Screen

 

While this information is optional it is important to fill in as much as possible. When implementing the Service Integration Matrix (SIM) accurate Director and Contact information will be critical to inter/intra agency communications.  When printing the SIM Referral Letter the contact information is one of the merged fields.  The Director and Contact Information does not have to match.  The Contact Name/number should be the person that a client would connect with or another User would connect with for more information about program services. 

Enter Contact Information by accessing the ADD  Button on the right hand side of the Contact Information Box. 

Checking the GENERATE APP NO? box directs the system to automatically assign the next application number each time a User is entering a new application.  Brand new systems should enter a "1" in the NEXT APP # box for all Intake Hierarchies.   Then the system will automatically generate an application number for a new Client Intake. 

Enroll Hierarchies in the Program Information Tab

Programs that should be available in the enrollment window should be assigned in the Enroll Hierarchies box. 

  • Click the ADD  Button to access the screen to select the hierarchies you want to assign.

Let's look at the working screen to assign hierarchies for enrollment.

  • Hierarchy - select the hierarchy through the Hierarchy  Button
  • Funded Slots - enter the maximum number of enrollment slots the system should interact with for the program
  • Start Date - start date for program enrollment
  • End Date - end date for program enrollment
  • SAVE
