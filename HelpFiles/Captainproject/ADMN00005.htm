This screen allows you to complete a User Search to view/update an existing User or create a New User Account . Users are required to enter their User ID and Password when they first launch the application.

Search for an existing User

Enter the criteria you want the System to use to search and click the search button.

 

While in search mode the full screen is in view only mode. You can select each tab and review User access and privileges.  In order to work in the screen or any of the tabs you must select ADD or EDIT   at the very top of the tab first.

Reset Password for an existing User

Users can change their password whenever they want.  System Administrators can also put a process in place for when they wish Users to update their passwords, if at all.

Here are the standards in place in the system for changing your password:

  • At least 8 characters long
  • At least one UPPERCASE character
  • At least one special character
  • At least one number character

Here is an example of a password:  DOG3456cat!

The image above shows you where the Change Password hyperlink is in the very top right corner in your browser.  Clicking that link will present the working screen to change your password.

  • Complete the fields
  • Click Change Password
  • Make sure you write your password down for future reference!

 

FORCE All Passwords change system-wide

As a System Adminmistrator you have full control over the security of your active User Accounts in the system.  You can FORCE All Users to change their password in just one click!  Let's look at the image below to find the FORCE AllPassword Button.

 

This is a very powerful button and there is no turning back!  If the system administrator clicks this button and confirms the action ALL Users will be prompted to change their password at their next login.   While this is a very good security option it's good practice to identify to Users this is going to happen, or at the very least teach them why they might see the prompt and what they have to do in the event of a system-wide password reset. 

Once a FORCE All Password reset has been completed, the system will pass through 60 days and then prompt the System Administrator with the red exclamation mark to do this again.  It is not required.  The prompt just lets the system administrator know that it has been at least 60 days (could be more) since the last forced password reset system-wide.
