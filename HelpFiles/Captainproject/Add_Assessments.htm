Add Scales  into the Matrix/Scales Assessment Screen

When you select the Matrix/Scales Assessment Screen the application you are working will present Assessment data, if any, based on the filters the User selects in this working screen.

We'll look at a different image with assessment records in this section of the screen to work with or review, but first let's understand how to work with the screen.

  • Matrix - Select the Matrix you want to work with.  If there is only one Matrix designed for Users to work with, it will automatically default into the field.
  • Assessment Date - Filter the data you can review by selecting a specific assessment date.
  • Status - Filter the data by selecting a specific assessment status you want the system to present.
  • View Other Assessment Button - exactly that, select other assessments to review
  • Applicant Assessment Dates - this subscreen allows you to create the assessments, by date, that you will use to complete client assessment entry.
  • Show Benchmark and Score CheckBox - If the User checks this box the screen will present where the client falls within the Scale based on the answers given and the dimensional movement +/-

Complete Client Assessment Entry in the Matrix/Scales Assessment Screen

Locate and Click the Button to access the Client Assessment Entry working screen to complete the Assessment by responding to the questions.

  • Matrix - Select the Matrix you want to work with. 
  • Scale -  When you select the Scale in this field the questions will change in the bottom of the screen.  Users can stay in this screen and move through all the Scales one at a time.
  • Assessment Date - If more than one assessment date has been added into the application you are working with, you can select the date you want to work with.
  • Bypass Scale Check Box - check this box to bypass the scale from the assessment session.

Note that there is a pdf image in the top right-hand corner of this working screen.  You can print out the Scale you are locked on and use it as a working tool or reference tool. 

How do you respond to the questions?   This function is simple and straightforward. 

  • Left Mouse Click the Question you want to respond to.  (it turns BLUE)
  • Right Mouse Click on the SAME Question, that is now BLUE, and the system will present the Responses to the Question that you can review and choose from
  • Left Mouse Click the response you want to select
  • Move through the Scale responding to Questions
  • You do NOT have to save each individual response; however, if you do not save the responses before closing the screen all responses during that work session will be lost. 
  • You must click CLOSE to exit this working screen.  Clicking SAVE will save your work but allow you to continue working.

Once you do save your responses the system will immediately calculate the value of the responses and plot the Scale into the Benchmarks and present that in BLUE FONT in the top section of the working screen (above the working questions area).

As you respond to Questions, the system will present a DELETE  Button in order to remove it.  If you find you have seleted the wrong response, just return to the question and make the correct response and click save.  Be cautious and careful when deleting.  Every response has its place in the assessment measurement. 

Be certain that if you have to complete a new/updated assessment that you create a NEW assessment date and respond to the questions again.  If you just have one assessment that you update you will not be able to see the dimensional movement (+/-)  change for your client/household.
