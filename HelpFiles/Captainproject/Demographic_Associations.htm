Demographic Associations Tab

The Case Demographic Report manages several categories in the report template.  The qualifying responses that are selected for the report are driven from the demographic associations in this tab.  Qualifying responses are typically driven by the funder and based on contract information surrounding reporting. 

The ability to manage how the report aggregates the counts for all categories allows Users to enter data without any concern for how it will be used in a specific report.  All customization to the report template is managed here. 

Associations can be changed when a specific report demands without any impact to data entry Users that are inputting and saving Client Intakes.  This screen is very straight forward.  Select the category on the Case Demographic Report at the top of the screen.  Once selected the bottom of the screen will shift to display the Report Template categories on the left and the Agency responses on the right. 

  • Select the Category that you want to work with
  • Click the  button to activate the screen to make your updates.
  • Click SAVE.

Age Range From is used in fields where a specific age range should only be used when the system aggregates data for the report run.  Any age that does not qualify for the age range will be excluded from the report run.
