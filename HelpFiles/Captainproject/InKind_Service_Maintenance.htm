The top part of the screen allows you to input specific information about a volunteer /donor that is donating to your program.  Remember that InKind doesn’t have to be cold hard cash.  It is absolutely time and talent! 

Our programmers were busy at work establishing a way for a User to input information about any volunteer or donor one time.  That’s the top part of the screen.  Notice you can track specific demographic data right along with their name and address.  Super!

Now the bottom section of the screen is where you can track the kind of time or talent donation the volunteer/donor shared with you.  Check it out!  Let’s say you have a volunteer that has been donating their time and talent in your program for the last 5 years.  You can track the date and time of each event. 

Think about the top part as an index card with the volunteer/donor info and the bottom part as a historical list of what they did.

You will find a Master Search at the top of the screen.  You can search to find existing donor/volunteer records already in the system.  It is important to always search first to ensure you're not maintaining more than one donor/volunteer record in the system for the same person.

Add a Donor/Volunteer in the InKind Service Maintenance Screen 

To add a NEW donor or volunteer to the system click the ADD  Button at the top of the InKind Tab to access the working screen.

Look at this screen much like an index card.  It's holding all of the information about the specific donor/volunteer.  Once you have entered the demographic data you must collect, click SAVE to save the record in the system.

This record will continue on throughout the entire lifetime of the Donor/Volunteer.

  • Code - is a unique identifier you will use so that the system keeps each and every volunteer separate.  You may want to spend a little time considering what those identifiers will be.  You can use #1 the first time and go right on from there.  It’s completely up to you!  It is a numerical field. 
  • Donor Type - is a 100% customizable drop down field where you can have any donor type option to select that you need.  If the drop down option you need is not available contact your system administrator to have it added into the agency table codes.
  • Donor Name - Enter the name of the donor/volunteer you want to assign to this record.
  • Child First Name and Last Name - HS Programs specifically use these fields to associate the donor/volunteer to the child enrolled in the program.

All other fields are self explanatory.  Your System Administrator will be able to add in additional selection options in the agency tables codes for you, as needed. 

Add Services to a Donor in the InKind Service Maintenance Screen 

To add a NEW service to the donor click the ADD  Button to the very right of the donor name row to access the working screen. 

  • Service Type - select the service type from the drop down field that you want to assign to this donor/volunteer
  • Date - enter the date the service was provided

Complete all other fields as needed.  This is a universal screen and many of the fields will be head start specific to manage time, transportation and mileage donated to the program. 

Edit Services attached to a Donor in the InKind Service Maintenance Screen 

To edit a service to the donor click the EDIT    Button to the very right of the service row to access the working screen. 

You will receive the saved service data record and can make any changes you want here in the edit screen.  Click SAVE when you have made the updates/changes you need.
