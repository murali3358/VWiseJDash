This report is used for authorized deliverables.  The report is sent to the Vendor confirming authorizations are in the Control Card, and this is the list of applications authorizing delivery. 

All Runs for the day:  if there were multiple authorizations in the morning, afternoon, check this box to include ALL runs for the date selected. 

Authorization date:  the date you’re running the report, only.

Select Site:  if you want to filter to a specific site

WINFAX filename:  enter the fax number

Print Format:  1 report: gives the User ALL authorizations, ALL Vendors, run for that day.

1 report for Vendor:  creates a separate report for EACH Vendor.
