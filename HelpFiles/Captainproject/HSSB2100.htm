Child Lists/ Counts  Report Screen

Child Lists/Counts Reports offers 10 Report Options to run a report and review class lists and various demographic data records for your program.  You can resource Child and Parent data.

 

  • Report Option - selet the report option you want the system to use when running the report.  The box on the right hand side will tell you what demographic data will be included in the report.
  • Sequence - select the order of the data you want to review
  • Enroll Status - select the status that will reflect the children you want included in the report run.
