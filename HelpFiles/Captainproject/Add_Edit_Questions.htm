Add Questions  in the Matrix/Scale Definitions Tab

In Questions Maintenance you will complete three different tabs:

  1. Groups
  2. Questions
  3. Responses

You must complete these tasks in the order of the tabs.  You must create your GROUPS and then attach QUESTIONS to the Group you want.  After that you attach RESPONSES to your Questions.  It's easy if you tell yourself to move from left to right in the tabs. 

Add Groups in Questions Maintenance Screen in the Matrix/Scale Definitions Tab

 

 

This step will group your questions together.  The groups become a visual guide for the User during the assessment input. 

  1. Sequence
  2. Select the Type from the drop down field
  3. Enter the Group Description exactly as you want the User to see it.
  4. Click SAVE

Edit a Group in the Questions Maintenance Screen in the Questions Tab 

  1. Select the Group you want to change/update
  2. Click the EDIT  button to access the Edit Screen.
  3. Make your changes/updates
  4. Click SAVE
