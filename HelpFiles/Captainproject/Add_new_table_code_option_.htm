Add a New Table Code Option in Agency Table Codes Screen

Locate the row with the    asterisk on the left hand side.  This identifies the row you need to use to add in a new table code option.

  1. Double Click in each cell across the row and enter the specific information for the new table code option you are adding.
  2. Assign a unique identifier to this specific table code option .
  3. Assign the description you want to User to see in the drop down box when selecting the correct option.
  4. Once you check the ACTIVE box the drop down option will be available in all assigned hierarchies.
  5. If you check the default box that table code option will automatically load into the screen each time it opens.  Users will still be able to select other options that are available.
  6. The Hierarchies button allows you to associate the code to the hierarchies you want Users to be able to use it.    Clicking on this button accesses a sub-screen where you can select what you want.
  7. When you move off of the row, the system will identify you have successfully updated the table code option(s).

  Edit a Table Code Option

Locate the blue row with the arrow on the left hand side.  This tells the system which table code option you want to edit.

  1. Double Click the cell(s) you want to edit and type the new information you want.
  2. When you move off the row the system will identify you have successfully updated the table code option(s).  If you close the screen without moving off the new row you created, the system will not save it.
