This is a program that allows a data feed to the community action agency association data bridge system.  The only use for this screen is if the User wants to create a cumulative/specific date range file. 

Every night this feed runs on a scheduler and will create a file for the current day's work.   All this is set up and run from a third party vendor.  Users do not have to do anything. 

If you feel you want to run this report, please contact the community action association for more details.
