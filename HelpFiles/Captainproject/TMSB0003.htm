This report will present demographics, by site.    The township table and zip code files are working behind the scenes. 

Date Range  From:  Use the first date of the program year.  (August 1st) 

Date Range To:  Current date that the User is running the report.    Changing the from date will produce a demographic report from that date forward.  Using the first date of the program year ensures the report run includes all possible data records. 

Print Detail:   Gives details that support the aggregated counts in the report run.

Sort By:   Site Code/Client Intake or Town Code/Client Intake. 

Select Site or Select Town:  All or specific selection

Report Only Elderly and Report only Disabled:   All or specific selection.

Report:  Heat and Rent (Intake) or Only Direct to Vendor (deliverables and utilities), both combines both filters.
