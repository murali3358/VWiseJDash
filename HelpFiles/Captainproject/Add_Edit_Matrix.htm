Add a Matrix to the Matrix/Scale Definitions Tab

  1. You should not check the Active Check Box until you have completed the entire process of buildng your Matrix.  Users can see the Matrix once you have checked the Active Box.  You can return and check and uncheck the box as many times as needed while building the Matrix.  Then you can see how your building is coming along.  This check box allows you to disable the Matrix if you are making significant changes/updates.  Then Users will not be able to interact with the Matrix while you are building. 
  2. Name your Matrix by typing the Description in the description field.  Make it meaningful to your Users particularly if you have more than one.
  3. Select the method you want the Matrix to use:
    1. Response Average Method
    2. Outcome Criteria Method
  4. Select the option you want to use in this Matrix for Assessment Dates. 
    1. Applicant Assigned Assessment Dates - the applicant defined assessment dates selection will allow you to have different sets of assessment dates for each application.
    2. Matrix Assigned Assessment Dates - Matrix defined assessment dates have one set of assessmentdates for multiple applications.
  5. Enter the number of days you want between assessments in the Interval (In Days) box.
  6. Check the Show Benchmark Assessments box if you want to see the Benchmark Assessments by default.
  7. Click SAVE

 

Edit a Matrix in the Matrix/Scale Definitions Tab

In Edit   Mode the system presents the same screen with the Edit tag at the top.  In Edit   Mode you can make changes/updates to these fields.  All changes/updates will impact the Matrix in every location the fields are resourced.   It will overwrite any existing data.  Click SAVE to keep the changes/updates.
