Service Plans are accessed in the Service/Activity Posting Screen.  Service Plans manage custom designed Critical Activities and Milestones.  This first view only screen allows the User to find an existing service plan already designed in the system and call it up and update/change it using the EDIT function. 

Service Plan record rows with RED font are Service Plans that have not be validated yet.  Users will be able to see these but not interact with them.  A best practice that might work well for the system adminstrator is NOT validating a Service Plan until it is completed and ready for Users to work in.  Making an adjustment to a Service Plan after Users are working with it could cause confusion.
