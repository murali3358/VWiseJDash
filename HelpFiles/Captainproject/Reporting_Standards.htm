Report function/button system standards are available throughout the report screens. 

You will find them at the bottom of every report screen.  These buttons allow the User to perform different functions.  They take on the same role, for the User as Toolbars do throughout the Case Management Screens. 

This button is found in the Adhoc Report Screen.  It allows the User to clear ALL fields that have been added into the Adhoc Reporting Screen while designing a new report.  It erases all the field selections to that point and the User can start over again with a blank report.

This group of buttons allows the User to save a report they have just created.  This is available in all screeens. In template Reports a User will set filters and also specific date ranges.  Saving the parameters of a report allow the User to get those parameters during future work sessions without having to spend time designing the report over again. 

These buttons will generate and preview .pdf documents.  The system will present a different preview list of .pdf documents only.   This area acts the same as Generate Report and Preview Reports. 

How to Save Report Parameters

Saving report parameters allows a User to build a report once, and use it over and over again. Locate and Click the Save Parameters Button to access the working save parameters screen.

  • ID -  The identifier the system will use when working with this saved report.
  • Description - A brief, meaningful description of the report you are saving.  This description should ensure that you don't have to open the saved report to know what it is going to do for you.
  • Click SAVE to save the report in your User Account.  (no other User will be able to see it.)

How to Associate a Report to another User 

Locate and Click the Associate Button to access with working screen to associate a report to another User.  If you don't see this button in the screen that means you do not have the security required to share reports with other Users.  See your System Adminstrator if you believe this function would be beneficial for you to use.   

 

  • Select the Module that the report is saved within
  • Left Mouse Click to lock the system in on the report description in the top list box .  This will be the report the system is going to associate to another User(s)
  • Select the User(s) you want to give the report assocation to from the left hand list box.  You do this by left mouse clicking in the cell to the left of their UserName.
  • Type in the Associated Description you want this User(s) to see when they access the system. 
  • Click SAVE to successfully associate the report to the selected Users.  The system will deliver a msg box sharing the assocation(s) were successful. 

How to Delete a Report(s) in your own User  Account

When a User saves report runs, this report is saved in their own User Account.  Other Users cannot see them.  Managing this report/preview sub panel in the system updates the active User's Account only.

Let's look at the Report Preview/Maintenance subpanel.  This is the working screen to delete reports you no longer want associated to your User Account.

  • Left Mouse Click the report you want to work with
  • Right Mouse Click the highlighted report to access the action options.
  • Mark - tags the highlighted report for deletion
  • Mark All - gives the User the option to tag ALL reports in their account for delete with just one click
  • Unmark All - gives the User the option to remove all tags previously marked 

Once you have tagged the reports you want the system to delete, you click the DELETE button in the lower left hand corner. 

The system will prompt you one last time to ensure you intend to delete all the reports you have tagged.  See the image below.  A Yes response will PERMANENTLY delete all tagged reports.  There is no option to recover them after they are deleted. 

 

How to Download a Report to your PC

When a User wants to download a report from the system directly to the hard drive of the PC they are working on, the DOWNLOAD button accesses that function.

You can directly download the selected file (PDF/EXCEL/TXT) into your local PC's hard drive. 

Downloads go directly to the download folder on the PC.   Anything downloaded to the PC can be accessed without being logged into the system.
