The Vendor Maintenance table is used in the LIHEAP, Case Management, and Emergency Services Systems.

  • This screen, when launched, will load both active and inactive vendors into the list box with filter options for vendor types, fuel types.
  • The User has the ability to sort the list box by vendor code or vendor name.  Left Mouse clicking the list box titles will allow you to view alternate sort options. 
  • If the vendor is inactive the row will appear as red to designate the inactive status.

 

All data fields on the left side of the panel are standard vendor specific data fields. The fields to the right control different areas of the system.

  • Contact Name – is the primary point of contact at that vendor.
  • Phone – this is the main phone number for the vendor.
  • Parent Vendor – this field is made available for future needs for ‘bigger’ LIHEAP vendors that have smaller vendors in the CAPTAIN Vendor table.
  • Fixed margin- this field is used to flag that the LIHEAP vendor participates in the Harbor Pricing for reduced unit pricing for oil.
  • Electronic Transfer- This dropdown controls electronic file formats for Utility Vendors. The output format of the electronic file is controlled with this field and is usually defined by the CAP Systems developers.
  • 1099 Cd- This field was made available to this screen for future use.
  • EIN- This field was made available to this screen for future use.
  • Cycle Code- This field was made available to this screen for future use.
  • Format- This field was made available to this screen for future use.
  • Contract/Agreement Returned check box: This allows customers to flag the vendor record that the contract with the agency was signed and returned.
