Master Poverty Guidelines

This screen allows you to define the following master charts that will used by the system in order to calculate where a household falls on specific federal and state income charts used to determine program eligibility:

  • Federal OMB (mandatory chart)
  • State Median Income (SMI)
  • County Median Income (CMI)
  • HUD

The master poverty guidelines chart is used during income verification.  When the User verifies income, the income details are compared to the assigned master charts in program definition.  The calculatead % is then plotted on the income verification screen. 

Charts need to be updated by the System Administrator prior to the end date of the current chart built.  If the next chart has not been built and a User attempts to verify income the system will not be able to do that.  As soon as regulatory entities release the next guidelines the System Administrator should build it into the system.  While more than one chart can be built in the system, date ranges can not overlap. Note below an example of two consecutive Federal OMB Poverty Charts. 

  • Fed OMB Chart 1/22/2006 - 1/21/2007
  • Fed OMB Chart 1/22/2007 - 1/21/2008

If a regulatory entity does not release the next year's guidelines prior to the expiration date of the current chart the System Administrator should extend the date range of the current chart without building a new one.  Once the new guidelines have been released the System Administrator can update the dates.
