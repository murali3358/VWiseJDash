Zip Code Maintenance Screen

This screen is used to maintain the zip code database. You can add new zip code records and/or change existing zip code records as needed.  Zip codes should be selected by the User from the database.  Manually entering a zip code may exclude that zip code from important task the system does behind the scenes.  (See - Program Definition to eliminate Users manually entering zip codes by setting the appropriate switch.)

Ensure that ALL zip codes that will be used are included in the zip code database.  If a zip code is not available for selection by the User they should request that it be added to the database. 

  • Define specific information for the zip code
  • Associate the zip code to applications in the system that need to use it
  • Enter the Head Start School Start Date to ALL zip codes that will be used in the Head Start application.  This field determines accurate age calculations in several report options.

Search for an existing Zip Code in Zip Code File Maintenance Tab

 Enter the criteria you want the system to use to search and click the search button.  

While in search mode the full screen is in view only mode.    In order to work in the screen or with any of the zip codes you must select ADD  or EDIT first.
