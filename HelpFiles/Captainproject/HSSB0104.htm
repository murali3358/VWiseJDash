Enrollment Summary Report Screen

The Enrollment Summary Report allows the User to have a summary snapshot of program-wide enrollment.  This includes, withdrawals, 30 day window, and attended one day data. 

This report is a template report that the system generates based on the filter selections the User makes in the report screen.

  • Site allows the User to run the report on specific sites or ALL sites in the program.
  • Report Type - generates the report based on the filter selection.  Site Summary delivers counts based on the site.  Room Detail separates the counts by the classrooms in the sites.  Fund Detail By Room further separates the counts by fund.
  • Funding Source allows you to run a report on a specific funding source or ALL funding sources at once.
  • As Of Date - the report run delivers counts based on the as of date.  for example, if you want to see enrollment records for a specific month in preparation for the Enrollment Report you submit in the Enterprise System, you will want to use the last day of the given month you want to report on.
