Medical Emergency Screen

Medical Coverage should be checked to indicate the child has medical coverage.  Once checked, Private or State fields activate and one must be selected.

  • Private Button - once checked you will required to enter an insurance plan/case number.
  • State Button - once checked you will be required to enter an insurance plan/case number and an insurance category.
  • Insurance Plan#/Case# - enter the private insurance number or state case number.  Enter a standard default response if you don't know the correct number.  Consider using a standard default entry so that in a report it stands out that the information has not be received.
  • Insurance Category - this only applies if it is a state plan, and it will be a required field.  The drop down option can be customized to the selections your program wants to use.
  • Insurance Name - Enter the name in the field.
  • Complete Doctor Name, Address and Phone as known.

Dental Information should be checked to indicate the child has medical coverage. 

  • Insurance Plan - enter the insurance plan number.
  • Insurance Name - Enter the name in the field.
  • Complete Dentist Name, Address and Phone as known.

Special Conditions is the area where you can maintain additional information about the child. 

If the child in the application has a diagnosed disability, check the box and enter the information as known.

In the Any Other Medical Concerns or Suspected Conditions box, left mouse click to select the category you want to work with.  Once you have selected the category (let's use the example ALLERGIES as above) then you can type in the bottom box what you need to.  Be careful to attach the information to the correct category by always confirming you have selected the one you want to work with first. 

The area above Emergency Contacts is Head Start specific information that is important to not overlook.  When entering an application into the system, it's important to add in the additional information here.   This is key information that interacts with other parts of the system based on what is selected.

  • Fund - select the anticipated fund support for the child.  It can be changed at enrollment if necessary.
  • Alt Fund - if there is an alternate fund, you can enter it here
  • Transport - select the correct response related to transportation for this child
  • Class Pref - if the family has a class preference you can select it here
  • Place of Birth - enter City and State
  • Pre-Enroll Client - check the box if you want to be able to pre-enroll the client
  • Repeater - if this is the first year you are using CAPTAIN, check the box if the child is a repeater.  After that the system handles this internally
  • Next Year Preparation - check this box if you have entered the child and want him/her to hit the NYP Report for planning the upcoming year. 

Enter the address the child will be picked up at on the bus.  Enter the address the child will be dropped off at the end of the day.  This data will present on the Bus List Report.

Don't forget to click SAVE!
