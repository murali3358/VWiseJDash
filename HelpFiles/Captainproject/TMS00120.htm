The Reserve Schedule allows the User to close off a date/time/slot in the appointment system.

This is used AFTER appointment templates are set up.   Reserved Slots create an exception to the template and will close off the date/time/slot with a specific reason.   

After creating a Reserve Slot, the next time the User launches the appointment system, the change will appear. 

 

  • An * in the last column will indicate that there are scheduled appointments for that slot and the system will not allow the User to close/reserve that specific slot.
  • The reason field has a limit of 20 characters.

 

This is the primary screen for Users to post appointments. When the screen is launched it will always default to the current period of time.

The Search area allows a User to select a specific intake site and period of time so that they can view the specifics.

If the site is left blank the system will give a high level view of the appointments for that period of time that is defined at the very top of the screen.

 

In order to open a specific day just double click on the row and the details of that day will appear.

Booking an appointment to a specific time slot:           

  • Highlight the time and slot for that day (row) you want the system to work with
  • Press Appointment
  • Fill the required fields at the bottom area of the screen panel

The Search button next to the SSN# field will allow the system to search the main client data base to see if that client is in the database so that the demographic data can be pulled into the panel to expedite the posting of the appointment.
