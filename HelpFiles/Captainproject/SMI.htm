State Median Income Chart (SMI) in Master Poverty Guidelines Screen

The top section of the SMI Tab displays the existing income charts already built in the system.

  • Define the SMI chart date range that you want to build in the row with the  in the far left position.
  • Define the percentage thresholds that you want to be present in the master chart. (30, 50, 80, etc.)
  • Click the hierarchies button to associate the chart you are building to specific hierarchies.     

The bottom section of the SMI Tab is where you input the percentage thresholds data for the chart.

This data you are inputting comes from the state specific chart that your agency is already using.  The income increments are used when comparing the income details for an application in the income verification process. 

The blue row is where the system is focused.  Double click in the cells to input the income increments for the percentage thresholds that have been designed in the top section of the SMI Tab. 

*Note - the delete buttons are greyed out.  They will activate with an option to delete a chart once income increments have been added to the chart.  You should never delete a chart that has been used.
