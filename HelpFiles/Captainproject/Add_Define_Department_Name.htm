Add (Define) a New Department Name

The hierarchy Tab presents all available Agency Names that are available to add departments to.  Let's look at an image of that.  Below we can see there is one Agency Name to work with. 

On the far right hand side of the Agency Name row you will find the .  Click this button to open the Add Department Screen below.

  1. Assign a unique code
  2. Type in the name of the department as you want it to display in the System
  3. Type in a short name for the dept
  4. Check the Alt Intake button if you want this department to interact with the SIM
  5. Click SAVE

You will now find a row in the Hierarchy Definition tab with the department information you have just saved.
