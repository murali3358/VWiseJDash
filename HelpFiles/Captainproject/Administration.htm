This area of the system handles all of the adminstrative tasks that need to be performed to:

  • Design the Hierarchy structure in the system
  • Design Program Definitions
  • Create and Manage New Users
  • Design and associate Eligibility Criteria, by program
  • Create and Update Poverty Charts (Fed OMB, HUD, SMI, CMI)
  • Work with Master Tables Maintenance
  • Design and Maintain Zip Code Master File
  • Design and Maintain Site Master File
  • Design Matrix/Scales - a comprehensive Assessment Tool Feature

Access to this module should be limited to a small group of Users that are administering the system for your agency.
