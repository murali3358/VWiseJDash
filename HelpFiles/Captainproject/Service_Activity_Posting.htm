The Service/Activity Posting Screen has three distinct sections Users interact with:

  • Service Plans
  • Referral From/To Details
  • Contact Details

The full Tab displays relevant data attached to each section.  This is View Mode only.  You must access the specific section's working screen to ADD data or EDIT exisiting data. 

Service Plans Section of the Service Activity Posting Screen

This section of the Service/Activity Posting Screen allows you to manage existing Service Plan(s) for any household member in the application.  Let's look at an image of this section.

****Notice the Client Intake Tab is still active.  The System follows you as you move through the screens.  Were you to return to the MainMenu Search screen to select a new application, all of the tabs that are open will immediately back-fill will relevant data associated to the newly select application.

The ToolBar system at the very top of the screen is the toolbar that you work with to add or edit existing Service Plans. 

Let's look at the row of cells that display Service Plan data in this section:

  • Code - this is a unique code that the system uses when interacting with the data
  • Service Plan Description - this is a brief, meaningful description that your system administrator has used to indentify the service plan
  • Site - site location assigned to the Service Plan
  • Caseworker - the selected staff working with the service plan
  • Start Date - when the Service Plan Started

Referred From/To Details section of the Service Activity Posting Screen

This section of Service Activity Posting allows you to keep track of all of the referrals to other service providers or referrals that you have received from other service providers. 

This feature will interact directly with the Agency Referral Database .  This database is 100% customized by your System Administrator.  If you do not find a referral agency/resource that you need, contact your System Adminstrator and request that it be added in.

This section of the Service/Activity Posting Screen allows you to  manage existing referral  for any household member in the application.  Let's look at an image of this section.

You can EDIT any referral by clicking on the EDIT  Button.  Once you have returned to the Agency Referral selection screen you can change the date, the FROM/TO field or uncheck the agency referral selected and then select another one.

Deleting a referral is permanent and that data cannot be recovered.

Contact Details section of the Service Activity Posting Screen

This section of Service Activity Posting allows you to keep track of all of the Contact(s) you have had with household members in the application. 

Contact with your client can take on many forms.  Programs determine what kind of Contact is going to be tracked in the system.  Here are just a few examples of contacts that can be tracked:

  • Phone Call
  • Office Visit
  • Home Visit
  • Work Visit
  • Support Group
  • Extended Case Management Phone Meeting
  • Case Management Meeting
  • Face to Face Crisis Intervention

You can track any type of contact you need to.  Contact your System Administrator if the How/Where drop down doesn't have the selection you need. 

Locate the Contact Tool Bar to access the working screen. 

This view mode allows the User to see any Contacts that have already been attached to the application.  This area displays:

  • The date of the contact
  • Which member of the household the contact was with
  • Where the contact took place
  • Length of Time
  • Primary Language
  • The staff member that had the contact with the household member
  • If there are addtional case/program notes to read there will be an asterisk in the Notes cell.  (Access the Case Notes image in the tool bar to view them)
