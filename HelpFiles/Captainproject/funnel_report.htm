Funnel Report Screen

The Funnel Report asks the system to run aggregated counts on the service plans that are selected with qualifying critical activities and milestones within the established report date range.  This report allows a User to review duplicated counts of services and outcomes by Service Plan. 

  • Service Plan Grid Box - Select the service plans in the top left hand corner of the sreen that you want the system to load into the grid box for reviewing aggregated counts.  To make those selections you left mouse click in the box to the left of the service plan description.
  • Caseworker - use this filter to see only service counts related to a specific caseworker
  • Date Range Fields - use these fields to set the date range parameters during the report run
  • Enroll Program - select the filter you want the system to use during the report run
  • Generate Grid - click this button to have the system generate the grid below of the aggregated critical activity and milestone counts.

Generate Grid in Funnel Report Screen

Click the Generate Grid Button to load the grid box with the data records based on the filters you have set in place.

  •  Depending on the number of Service Plans you checked in the upper right hand corner you will be able to scroll through all the service plans to review the aggregated counts for all critical activites and milestones.
  • Show Details Check Box - check this box to access the data records that support the counts in each grid box.  Highlight the critical activity or milestone you want to review and check the Show Details box.  The screen will split and the lower half will display the records.
