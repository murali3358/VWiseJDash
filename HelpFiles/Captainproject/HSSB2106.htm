Child Track List Report Screen

Child Track List Report allow the User to generate a report that displays client tracking data attached to child applications. 

 

  • Report Sequence - how the report is sorted in the report run
  • Applicant Number - if you are running this report to review one specific child's data, you will be able to BROWSE for the child if you don't readily have the application # to type into the field.
  • Sites - which sites the report will include in the report run
  • Tracking Comp Code - This field filters data to the specific component area you are focusing on for your report run. 
  • All Task vs Selected Tasks - selected tasks is an additional filter to minimize the amount of data that will be included in the report run
  • Show Task - select the filter that you want for your report rn
  • SBCB Dates Before - checking this box and entering a date in the field is a filter that will only include tasks that have a SBCB prior to the date in the field.
  • Active/Inactive - Select the filter that you want to be used in the report run. 
  • Children Age - leave the default setting unless you want to filter the report to a specfic age, by year.
  • Base Ages on - select the filter you want to be used in the report run.
