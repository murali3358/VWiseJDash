Add a New Zip Code

1. Click  ADD   at the top of the Zip Code File Maintenance tab to activate ADD mode.

2. Customize the zip code by entering the zip code specific information in the fields as needed. 

3.  Head Start School Start Date - is the Month and Year that is used for the Kindergarten start date cut off.  Various functions in the system resource this data in order to calculate accurate age requirements in Head Start.  To avoid the system tagging reports and data result with NO AGE, it is important to complete this information on EVERY Zip Code entered into the system.  Select THIS PROGRAM YEAR in the Year Field if this is the first year you are using the system. 

4.  Click SAVE.

 

Edit a Zip Code

1.  Click  EDIT  at the top of the Zip Code File Maintenance tab to activate EDIT mode.

2.  Edit the zip code by changing the zip code specific information in the fields as needed.

3.  Click SAVE.

 

 Delete a Zip Code

 1.  Click  DELETE  at the top of the Zip Code File Maintenance tab to activate DELETE mode.  You MUST remove all associations/access  to screen functions/reports to all applications before the system will allow you to delete. 

2.  The system will ask you to confirm you want to delete the zip code from the system. 

3.  Click OK to complete the deletion.  Once this is completed all reference to the zip code is gone and it cannot be recovered.
