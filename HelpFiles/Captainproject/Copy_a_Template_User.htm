A specific User can be designated a Template User in order to use it to assign those privileges to other Users. This allows you to create a complex User and globally assign that to other Users in the future. 

There is no permanent connection between Users when this feature is used.  Once you copy the template Users privileges the connection is broken.  Changes made to a Template User in the system  only impace that specific User account.

This is also beneficial when you have to create a new User that has very complex privileges yet not the exact privileges of an existing template User.  You can copy the User and remove the few privileges you don't want the User to have.  You're capitalizing on the time savings the opposite way!

Copy a Template User

  1. Go to the Assign User Account & Privileges Screen
  2. Search for the User you want to work with
  3. Click EDIT at the very top of the tab
  4. Click to COPY FROM button
  5. Select the Template User you want to copy
  6. Click SAVE 

 

Reset a User Password 

Once a new User Account has been saved, the Copy From button changes to RESET PASSWORD button. 

Sometimes a User asks to have their password reset.  Perhaps they forget what the password is.  This reset button resets the password back to the UserID.  Once the User logs in the system will prompt the User to change their password.
