Add Reasons in the Matrix/Scale Definitions Tab

Reasons are selected by the User to indicate the reason why there was a positive or negative (+/-) dimension change to a specific Outcome. 

  1. Click the ADD  button on the lower right hand corner of the screen
  2. Type the new description for the reason you are adding
  3. Select positive or negative in the P/N field
  4. Click SAVE

 

Edit Reasons in the Matrix/Scale Definitions Tab

  1. Select the Reason you want to change/update
  2. Click the EDIT button to access the Edit Screen.
  3. Make your changes/updates
  4. Click SAVE
