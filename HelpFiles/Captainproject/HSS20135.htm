Site Schedule Screen

Before working with Site Schedules all the Head Start Sites must be built into the Master Site Table in System Administration. 

Site Scheduling is the screen where the system administrator designs the open/closed program services days for the HS/EHS PY Calendar.

The Site Schedule, based on a specific program year, will communicate to:

  • Attendance Posting Screens
  • ADA Reporting

The Site Schedule, based on a specific program year, will tell the Attendance Posting Screens what days are open. If the Site Schedule is not built correctly, Users will not be able to post daily attendance.

All Site Schedules need to be set-up after each year's System Rollover has been completed.   This step in designing and creating each year ensures that each program year's center/classroom structure is properly reflected in the system.

There is one Master calendar in each program year for each month.  These calendars should be based on when the majority of the clients will be attending class.  Whichever program option has the largest start/end date range should be your Master Calendar. 

Once that calendar has been set-up and attendance has been posted they system will not allow a User to delete the calendar.  A User will, however, be able to update/edit the calendar.  If you're not sure if there is attendance posted for a particular calendar, run a Deatils Attendance Report to see if records are attached.

Any Site Schedule Calendar that is designed AFTER the Master Calendar will be considered an exception calendar.  The Master is available for all SITES, all MONTHS, unless an exception calendar has been created that will override the Master Calendar.  This allows SITES or ROOMS that have different program services days to post attendance according to their own calendar or even FUND specific calendars.

*NOTE - Please do not set up calendars that are not necessary.  There should only be calendars set up if a Site, Room, or a specific funding source is doing something other than the Master Calendar for program services days.  

Sort By in Site Scheduling

Locate the Sort by field at the top of the screen. 

You can sort the schedules to make it easier to locate the specific calendar you want to work with.  Select the drop down option in the Sort By field that you want to use.

 

Add a specific monthly Calendar to Site Scheduling

Locate and Click the  button in order to acces the working screen to add a new calendar to your program year.

  • Hierarchy selection is the hierarchy position the User selects to complete tasks that are connected to your site schedule
  • Room - select this if you are building a room specifiic schedule
  • Site - select this if you are building a site specific schedule
  • Fund - sleect this if you are building a fund specific schedule
  • Master - select this if you are building a master scheduel (one per month/per program year)
  • Site - select the site you want to build the schedule for
  • Room - select the room you want to build the schedule for
  • AM/PM - select the AM/PM designation for the room you are building a schedule for
  • Funding Source - select the specific funding source supporting services for this calendar (only available if you have already checked the FUND box)
  • Month - select the calendar month in the program year that you want to create
