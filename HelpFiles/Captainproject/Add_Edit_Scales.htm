Add a Scale in the Scales Maintenance Screen found in the Matrix/Scale Definitions Tab

Scales can be associated to just one Matrix, or more.   

  1. Matrix:Select the Matrix in the Matrix field.  This is the Matrix the system will move forward with all associations.
  2. Sequence: Assign a sequence number.  This will determine the rotation of the Scales and how they are displayed to the User.
  3. Description: Type in the question you want the User to see.
  4. Rationale:  Type in the rationale you want the User to see.

Assessment Override Check Box - add description

Always remember to click SAVE.  If you don't click SAVE, everything you have done will be lost.

 

Assign the Scale to the hierarchies where Users need to work with it in the Matrix/Scale Definitions Tab

On the right hand side of the Scales Maintenance Screen you will find the hierarchies box.  Users that have access to the hierarchies that appear in this box can see and/or work the associated Matrix. 

Click  to access the sub-screen where you can select the hierarchies you want this Scale associated to. Don't forget to click SAVE when you are done. 

Edit a Scale in the Matrix/Scale Definitions Tab

By selecting the Scale and then clicking the EDIT  button you can make any adjustments you want to your work.  Sometimes it's a typographical error or maybe you want to change a hierarchy assignment, or add an additional one!

All of the information you designed when you added the Scale will populate in the fields and you can make your updates. 

 

Edit the hierarchies where Users need to work with it in the Scales Maintenance Screen

  1. Click the Hierarchies  Button to update the hierarchies where a User can access this Scale.  On the sub-screen this edit is a toggle switch check box.  Uncheck the box to remove the assignment.  Check the box to add the assignment. 
  2. Click SAVE
