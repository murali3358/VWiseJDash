Waiting List Report Screen

Use this template report to generate a waiting list for your program. 

  • Report Sequence - filters the report to generate records in Site/Name order or Rank order
  • Ranking Category - filters the report to generate on all rankings or selected rankings
  • Inlcude Case Notes - if the box is checked the report run will include Case Notes
  • Funding Source -  selected funding allows you to generate a report for specific funding source or all funding sources
  • Age - the default (0-999) will generate a report that includes all children all ages.  Changing the age range has the potential to change the children included in the report run.
  • Site - using this filter you can generate a report on a specific site or classroom or generate a report for All Sites
  • Income Status - select the report filter that you want to review in the report.  Select all to include all income status conditions.
  • Include Clients - this filter allows you to see all children placed in a wait list status, children that have not been placed in a wait list status or both.
