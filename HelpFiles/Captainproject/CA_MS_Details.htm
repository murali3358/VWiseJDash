Add/Edit  CA/MS Details to a Service Plan

Once a Service Plan has been added into the system you design the plan by associating Critical Activities and Milestones Users will work with.  Highlight the Service Plan you want the system to lock on.  (Left Mouse Touch it, Turn it BLUE)  then Locate and Click the EDIT  Button at the top of the Service Plan Admin Screen Tab to access the working screen.

 

  • Select the BRANCH you want to work with
  • Search Text allows you to use key words to find a CA or a MS quickly for selection
  • Results List Box is where you select the results responses the User has access to in the Service Plan
  • Funding Source Box is where you can select the funding sources that will be available to the User
  • Legal Issues is where you can select the response options the User has access to in the Service Plan
  • Don't forget to click Save SP Details button to save your work!

How do you select each of these?  Note the green check marks above.  Throughout the screen when you want to select a row, left mouse click in the cell to the left of the record description and the green check mark will appear.  If the green check mark is there and you want to remove the select just do the same thing!  Click it and the row will be deselected and removed from the Service Plan.

Move CA/MS rows to change the order they appear in the Service Plan 

You can redesign the order of the CA and MS rows in the Service Plan.  Select the CA or MS you want the system to work with.  Right Click on that row to access the move function and select the new location in the plan.  Make certain you SAVE your updates.
