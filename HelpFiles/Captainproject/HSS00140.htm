Bus Client Placement Screen

This screen is used to assign children to a specific bus and route.  When the User selects the Route in the top list box, children assigned to the bus route will load into the bottom list box.   The system standard ADD button above the screen tabs will allow the User to access the working screen to assign a child to the Bus/Route.

Here is the working screen to assign a child to a Bus/Route:

  • Search and Select the application number for the child you want to assign to the Bus/Route
  • Pickup and Dropoff times that are specific to the child should be entered here.
  • Enter any comments that will be helpful to note.
