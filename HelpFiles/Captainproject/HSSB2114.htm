Site Track Task to Address Report Screen

Site Track Task to Address Report will allow the User to review client tracking tasks that have not been completed for children you are montoring.  You can set filters on the report to minimize the data you will receive in the report run.  

  • Report Sequence - select the filter option you would like presented in the report
  • Applicant Number - enter the application number if you're running the report on a selected applicant.  If you're unsure of that number, click BROWSE and search the system for the application.
  • Site - the default is ALL Sites.  Use Multiple Sites/Classes if you want to filter to a specific center/classroom.  You can select more than one.
  • Tracking Component - if you want to focus on one specific component area you can select it here.  When selected multiple tasks the sub-screen will only present tracking questions from that component.  This field gives the User the ability to not have to look through all tracking questions to find what they want.
  • SBCB Dates -  the system focuses on a should be completed by approach that is based on the date entered.
  • Active/Inactive - the system will generate the report based on the filtered selection.  Both ensures a User they see all potential data. 
  • Enroll Status - the system will generate the report based on the filtered selection.  Both ensures a User they see all potential data.
  • Funding Sources -  they system will generate the report based on the filtered selection. 
  • Children Age - the default setting (0-999) ensures that all children, regardless of their age, are considered in the report run.  If you want to filter to a specific age (by year) you should set that filter here. 
  • Base Ages On - the system will generate the report based on the comparison between the child's age and the selected filter.  Kindergarten Start Date has been set in Zip Code File Maintenance to automatically calculate the Intake Age, Current Date Age, or Kindgarten Age.
