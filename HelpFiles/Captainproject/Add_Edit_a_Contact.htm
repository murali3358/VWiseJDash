Add a Contact to the Contact Details section of the Service Activity Posting Screen

 

Locate and click the ADD  Button in the Contact Details Tool Bar above to access the working screen. 

Let's look at an image of the working screen. 

Complete the fields that are set by your system administrator.

  • Contact Date - date of the contact
  • Contact Name - select the name of the household member you had contact with
  • CaseWorker - who had contact with the contact name
  • How/Where - select the correct option
  • Bill To - select the agency that needs to be billed
  • Bill Unit - select the appropriate billing unit
  • #of Contacts - enter the number
  • Duration Type - select either Duration or Type and then complete the fields
  • Language Speaks - primary language
  • Interpreter Needed - select the correct response

Don't forget to click SAVE to attach your data entered to the application. 

Edit a Contact to the Contact Details section of the Service Activity Posting Screen

 

  • First select the row in the Contact Details list box that you want to edit.  You do this by left mouse clicking on the row in the list box you want to work with.
  • The system identifies the row you have selected by moving the arrow on the far left hand side of the list box AND by highlighting the row BLUE.  
  • Once you visually confirm that you have selected the contact you want to edit, lo cate and click to EDIT  Button in the Contact Details Tool Bar above to access the working screen. 

Let's look at an image of the top of the Edit working screen.  Note it now says Edit Contact.  The system will present the same screen fields as in ADD mode and you will be able to update or edit exisitng data.

 

Delete a Contact to the Contact Details section of the Service Activity Posting Screen

 

Locate and click to DELETE  Button in the Contact Details Tool Bar above to access the working screen. 

This function is quick and unforgiving.  You cannot recover any data that has been deleted from the system.  There's no nevermind mode.  The system immediately asks you - are you sure you want to delete?  You click OK.  That's it!  Nothing more.  It's gone. 

Let's be certain you understand how the system does this.  Once you click the delete button the system is already locked on one particular entry.  Look at the image below to see that. 

See the blue highlighted row in the Referred From/To Details screen that is behind the Delete msg?  That's the entry that will be deleted.  Make it a habit to always confirm what is being deleted BEFORE you click YES.

Deleting a contact is permanent and that data cannot be recovered.
