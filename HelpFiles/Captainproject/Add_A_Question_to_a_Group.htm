Select the Eligibility Group Description you want to work with.  The arrow on the left hand side and the blue highlight will indicate to the User which description (if there is more than one) they system is locked on.  This is the description that Questions/Responses will be added to.

 

Locate and Click the ADD  button on the right hand side of the blue highlighted Eligibliity Group Description to access the working screen to add a question to this group.

  • Select the Question in the drop down field that you want to add to the group description.
  • Select the members in the application that should respond to the question.
  • Set the conjunction you want for the question.
  • Click SAVE
