This screen is used to build available slots for scheduling appointments.  If an Agency has multiple intake sites/locations that will take appointments a master must be set-up for each location.  There are 3 types of templates that can be created.

1-      Master Template for each Site

2-      Site/Month Template for a particular site and month

3-      Site/specific date for a particular site and date

Once calendars are created and saved and changes need to be made press the “browse” window to selected the applicable calendar.  I f appointments have already been scheduled the system will not allow the User to make any changes; however, the system will allow the User to open a closed spot.

 

 

A ‘master’ calendar must be set-up for each location if they operate differently (even if there is only one site..that one site must be setup).  The master will be used for ALL the months unless an exception has been created for a specific month or a specific date for the location.

To set-up a Master Calendar:

1.       Click on the “add” button in the toolbar.

2.       Click on the ‘search’ button in the site field.  Select the applicable site from the Site Selection window.

3.       In the Template Type area the system will always default to the “master” radio button unless there is already a master for that site.   

4.       In the Time Slot field the system will automatically default to 3 slots.  Meaning that there will be 3 workers taking appointments.  So if your Agency has 4 workers change the 3 to 4.

5.       In the Period in Mins field the system will automatically default to 20.  Meaning you are allowing 20 minutes to complete an application (for each worker).  If this is not correct click on the down arrow and make the necessary changes.

6.       You will also notice that the system automatically has Monday, Tuesday, Wednesday, Thursday and Friday “open”.  And that Saturday and Sunday are closed.  If there is a day (Monday – Friday) that your Agency never takes appointment please mark that day as “closed”.  You do this by clicking on the applicable radio button.

7.       You will also notice that the screen has 2 unique columns. The 1st column starts at 12:00 am and ends at 11:40am.  The 2nd column starts 12:00 pm and ends at 11:40pm.  So based on your appointment schedule make the applicable changes to the times that are marked as closed and open.  If a time is marked as ‘open’ and it should be marked as closed then double click on that time slot.  For example you can see that the system as from 8:00 am open…ready to take appointments.  Your Agency may not make appointments until 9:00 am.  Therefore you would need to mark from 8:00 am until 8:45 am closed.  If your Agency does not take appointments during the lunch break you would need to mark that time as closed. And in the left column you will notice that you can take appointments until 8:45 p pm.  So remember to close all the times that are not applicable before you press the “save” button (in the toolbar).
