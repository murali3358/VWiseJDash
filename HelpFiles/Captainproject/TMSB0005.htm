This report gives a list of everyone who has been approved as a utility client, by Utility Vendor only.

Re-Run:  Yes, new screen re-run box to find the existing run they want to use.

Date field is no longer activated, click Re-Run Dates Button to select a previous run.

 

Re-Run Date:  used only if Re-Run YES filter is in place.

Vendor:   can select one Utility Vendor if necessary

Utility Notification Date:  the default is the current date.  The User can change the date if they want.

 

All Clients:  Picks up every application, including previous runs.   The first time the report is run in the program year would probably be the only time to use this filter.

Clients since last run: will only pick up the applications that were not included in a previous run.
